Пользователи, лицензии и глобальная тема сайта. Данные приходят с Node API того же проекта — откройте профиль через http://localhost:PORT/profile.html, если таблицы пустые.

Тема сайта

Акценты для тёмной и светлой темы (#RRGGBB), видны всем гостям после перезагрузки страниц.

Лицензионные ключи

Срок ключа задаётся суммой дней, часов и минут (хотя бы одно значение > 0). Количество — сколько ключей создать за раз.
